Você está cansado de construir sites que carecem de desempenho e foco em conteúdo? Não procure mais do que o Astro 2.0, a última versão do popular framework web que está mudando o jogo.

O Astro 2.0 chegou e está trazendo novos modos de construir sites desempenháveis e focados em conteúdo. Com recursos como Content Collections e Hybrid Rendering, você pode melhorar sua experiência de desenvolvimento e criar sites de qualidade.

Uma das principais melhorias do Astro 2.0 é a segurança de tipos completa para Markdown e MDX. O parsing, validação e geração automática de tipos TypeScript tornam o trabalho com Markdown na web fácil. Você não precisa mais se preocupar com coerência e propriedades frontmatter ausentes, pois a API Content Collections do Astro 2.0 cuida de tudo para você.

Mas não é só sobre Markdown, o Astro 2.0 também oferece Hybrid Rendering que combina o melhor do conteúdo estático e dinâmico. Isso permite um melhor desempenho de renderização para páginas populares, uma melhor performance de construção para sites de grande porte e até mesmo a possibilidade de adicionar uma API ao seu site estático existente.

O Astro 2.0 também inclui sobreposições de erro redesenhadas, um servidor de desenvolvimento melhorado e uma atualização para o Vite 4.0.

Não perca a oportunidade de levar seu site para o próximo nível. O Astro 2.0 está disponível no npm agora, atualize seu projeto existente ou comece um novo do zero com astro.new. Saiba mais sobre as novas funcionalidades e melhorias e descubra como o Astro 2.0 pode ajudá-lo a criar um site melhor.
